# dotfiles

团队的基础系统配置，目前只是用于 mac 系统

## 准备工作

在开始安装之前，需要关闭系统的 SIP 服务，[参考](https://sspai.com/post/55066)

## 安装

```sh
git clone git@github.com:36node/dotfiles.git
cd dotfiles
./install.sh
```

## 收尾

重新开启 SIP 服务
